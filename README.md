# Bug Tracker

## Setup

### Installation

-> Download the ZIP file

-> Install the dependencies (package.json)

-> Fill the .env.exemple file

### Running

-> Open the terminal

-> Type in `npm run dev`

- React (Front-end)
- NodeJS (Back-end)
- Express (NodeJS Framework)
- MongoDB (Database)
